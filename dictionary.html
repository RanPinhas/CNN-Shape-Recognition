חזור לעמוד המודל

מילון מושגים ברשתות נוירונים (40 מונחים)

1. AI (בינה מלאכותית) - תחום במדעי המחשב העוסק ביצירת מכונות המסוגלות לבצע משימות הדורשות אינטליגנציה אנושית.
2. CNN (רשת קונבולוציה) - רשת נוירונים המיועדת בעיקר לעיבוד וזיהוי תמונות.
3. Neuron (נוירון) - יחידת החישוב הבסיסית ברשת, המקבלת קלט, מכפילה במשקלים ופולטת פלט.
4. Weights (משקלים) - פרמטרים נלמדים ברשת הקובעים את עוצמת הקשר בין נוירונים.
5. Bias (הטיה) - ערך קבוע המתווסף לחישוב הנוירון ומאפשר גמישות בפונקציית האקטיבציה.
6. Forward Pass - תהליך העברת המידע קדימה מהקלט ועד לקבלת הניבוי הסופי.
7. Backpropagation - אלגוריתם ה"הפצה לאחור", המחשב את שגיאת המודל ומתקן את המשקלים כדי לשפר את הביצועים.
8. Epoch (מחזור אימון) - מעבר אחד שלם של המודל על כל סט נתוני האימון.
9. Learning Rate (קצב למידה) - גודל הצעד שהמודל לוקח בעת עדכון המשקלים בכל איטרציה.
10. Loss (שגיאה) - פונקציה המודדת את הפער בין הניבוי של הרשת לבין התשובה הנכונה.
11. Overfitting (התאמת יתר) - מצב בו המודל לומד "בעל פה" את נתוני האימון ונכשל על נתונים חדשים.
12. Underfitting - מצב בו המודל פשוט מדי ולא מצליח ללמוד את התבניות בנתונים.
13. Convolution (קונבולוציה) - פעולה מתמטית המחלצת מאפיינים מהתמונה באמצעות החלקת פילטר.
14. Filter / Kernel (פילטר) - מטריצת מספרים קטנה העוברת על התמונה לזיהוי מאפיינים כמו קווים וזוויות.
15. Feature Map - פלט של שכבת הקונבולוציה, המייצג את המאפיינים שזוהו בתמונה.
16. Pooling (פולינג) - פעולת צמצום ממדים השומרת רק על המידע החשוב ביותר.
17. Max Pooling - סוג של פולינג הבוחר את הערך המקסימלי מכל אזור בתמונה.
18. ReLU (Rectified Linear Unit) - פונקציית אקטיבציה המאפסת ערכים שליליים ומשאירה חיוביים כפי שהם.
19. Softmax - פונקציה הממירה את פלטי השכבה האחרונה להסתברויות (בין 0 ל-1, סכום 1).
20. Dense Layer / Fully Connected - שכבה שבה כל נוירון מחובר לכל הנוירונים בשכבה הקודמת.
21. Flatten (השטחה) - המרת מטריצה רב-ממדית (תמונה) למערך חד-ממדי.
22. Gradient (גרדיאנט) - וקטור הנגזרות המצביע על כיוון העלייה התלול ביותר של פונקציית השגיאה.
23. Gradient Descent (מורד הגרדיאנט) - אלגוריתם אופטימיזציה למזעור פונקציית השגיאה על ידי הליכה נגד הגרדיאנט.
24. Batch (אצווה) - קבוצת דוגמאות אימון המועברות יחד במודל לפני עדכון המשקלים.
25. Accuracy (דיוק) - מדד המייצג את אחוז הניבויים הנכונים של המודל מסך כל הניבויים.
26. Classification (סיווג) - משימה של שיוך קלט לאחת ממספר קטגוריות מוגדרות מראש.
27. Supervised Learning (למידה מונחית) - אימון מודל בעזרת דוגמאות מתויגות מראש (קלט ותשובה נכונה).
28. LocalStorage - זיכרון מקומי בדפדפן המשמש כאן לשמירת משקלי המודל.
29. Activation Function (פונקציית אקטיבציה) - פונקציה מתמטית הקובעת אם וכמה נוירון "יידלק" ויועבר הלאה.
30. Input Layer (שכבת קלט) - השכבה הראשונה ברשת המקבלת את הנתונים הגולמיים.
31. Output Layer (שכבת פלט) - השכבה האחרונה שמספקת את תוצאת המודל.
32. Hidden Layer (שכבה נסתרת) - כל שכבה ברשת הנמצאת בין הקלט לפלט.
33. Cross-Entropy Loss - פונקציית שגיאה נפוצה למשימות סיווג, המודדת הבדלים בין הסתברויות.
34. Stride (צעד) - מספר הפיקסלים שבהם הפילטר זז בכל פעם על פני התמונה.
35. Padding (ריפוד) - הוספת שוליים לתמונה (לרוב אפסים) כדי לשמור על גודלה לאחר קונבולוציה.
36. Vanilla JavaScript - כתיבת קוד JS טהור ללא שימוש בספריות או פריימוורקים חיצונים (כמו בתרגיל זה).
37. Canvas API - ממשק בדפדפן המאפשר ציור ועיבוד גרפיקה של פיקסלים בזמן אמת.
38. Normalization (נרמול) - שינוי קנה המידה של נתונים (למשל פיקסלים מ-0-255 ל-0-1) להאצת הלמידה.
39. One-Hot Encoding - ייצוג קטגוריות כוקטור של אפסים ו-1 בודד (למשל: [0,1,0] לריבוע).
40. Hyperparameters - הגדרות שהמתכנת בוחר לפני האימון (קצב למידה, גודל פילטר, epochs).
